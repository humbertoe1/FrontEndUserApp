Esta é uma aplicação feita por mim, Humberto Escorel. Ela foi criada com o objetivo de aprender mais sobre a integração do Frontend com o Backend. Vocês podem consultar/copiar os códigos aqui utilizados em meu repósitorio no GitHub. De maneira geral, é um simples CRUD (Create, Read, Update, Delete) de Usuarios. Sintam-se a vontade para sugerir qualquer mudança :)

Abaixo estão as tecnologias que eu decidi utilizar neste projeto!

Para o BackEnd eu utilizeri o .NET, com a linguagem de programação C#, ASPNET Core MVC e a EntityFrameworkCore para integração com banco de dados. Além do padrão de projeto MVC citado anteriormente, também utilizei o padrão Repository.

Já para o Frontend, eu resolvi aprender a utilizar a Framework Angular, com os pacotes do PrimeNG. É incrível como essas ferramentas facilitam a vida do desenvolvedor!

Utilizei um banco de dados em SQL com o auxilio do SQL Management Studio da Microsoft!
